Laboratorio Práctico

Evidencia completa del escenario: topología, configuraciones GUI, pruebas de conectividad y video demostrativo del enrutamiento estático.

Temas

01 · Fundamentos 02 · Topología IP 03 · Configuración CLI 04 · Interfaz gráfica 05 · Verificación 06 · Laboratorio

Objetivo del laboratorio

Implementar conectividad end-to-end entre tres redes LAN utilizando únicamente enrutamiento estático en tres routers Cisco 2911 dentro de Packet Tracer. Al finalizar, cualquier host debe poder comunicarse con hosts de las otras redes mediante ping y tracert.

Pasos realizados

  1. Diseño de topología con 3 routers, 3 switches y hosts en cada LAN.
  2. Asignación de direccionamiento IP a interfaces y hosts.
  3. Configuración de rutas estáticas en cada router (GUI y/o CLI).
  4. Verificación con show ip route y pruebas ping.
  5. Simulación de tráfico y documentación con capturas y video.
